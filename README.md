# auto_farm_slap_battles

# HOW TO USE

1. Copy the code inside script.lua.
2. Join Slap Battles and make sure you in the lobby not arena
3. Run the script with your executor
4. If you want to stop the script, you can rejoin with infinite yield

Enjoy!
